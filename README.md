# scoutswondelgem.be-node

```javascript
// Instaleer yarn
npm install -g yarn

// Installeer de nodige packages
yarn install

/** [Development] **/
// Slechts een keer runnen, tenzij je natuurlijk ook
// een script runt dat ergens tailwind:css:prod gebruikt
yarn run tailwind:css // Duurt lang
// Als je tailwindcss is gebuild, kan je dit gebruiken
// om development server te starten
yarn run serverstart

/** [Production] **/
yarn run start
```
